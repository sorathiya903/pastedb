Upload files or selected code to PasteDB directly from VS Code. Secure API key storage, dashboard, recent pastes, progress bar and more.

Install


ext install adityasorathiya.pastedb

Install from VS Code


🔎 Find it in the Extensions Marketplace

Open Visual Studio Code, click the Extensions tab (or press Ctrl + Shift + X), then simply search for "PasteDB". You'll find the official PasteDB extension published by Aditya Sorathiya. Click Install to start uploading code directly from VS Code.

🚀 Get Started in 5 Steps

①

Install Extension

Install PasteDB from the Visual Studio Marketplace or search for PasteDB inside VS Code.

↓
②

Generate API Key

Create a free PasteDB account and generate an API key from your dashboard.

↓
③

Open Dashboard

Open the PasteDB dashboard inside VS Code and paste your API key. Configure visibility, expiration and other settings.

↓
④

Upload Code

Right-click any file or selected text and choose one of the PasteDB upload commands.

↓
⑤

Share Link

Your paste is uploaded instantly. Copy the generated URL or open it directly in your browser.
