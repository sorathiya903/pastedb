How PasteDB Works

Create, share and manage your code, notes or images in just a few steps.

1. Create a Paste

Write code, notes or upload images. Choose between public, unlisted, private, burn-after-read or expiring pastes.

2. Secure Your Data

Enable browser-side end-to-end encryption for private and unlisted pastes so only people with the decryption key can read them.

3. Share Instantly

Share using a direct link, QR code or Nearby Share. Public pastes can also be discovered through the Explore page.

4. Access Anywhere

Manage your pastes from your dashboard, create them through the API, Python SDK or VS Code extension, and even run supported code directly from the viewer.

Who is PasteDB for?

PasteDB is a secure online pastebin and code sharing platform built for anyone who needs to share text, code snippets, notes or images instantly. Whether you're programming, teaching, studying or collaborating with others, PasteDB makes sharing simple and secure.

👨‍💻 Developers

Share code snippets, debug errors, create syntax-highlighted pastes, use the Python SDK, API and VS Code extension.

👩‍🏫 Teachers

Distribute notes, assignments, code examples and images to students using links or QR codes.

🎓 Students

Submit projects, store notes, share programs with teachers and classmates, and keep drafts safely.

🏢 Teams

Collaborate securely with End-to-End Encryption, expiring pastes, burn-after-read links and analytics.

🛠️ IT & Support

Share logs, configuration files and troubleshooting information quickly without email attachments.

🌍 Everyone

Share text, notes, images and files online instantly with public, private or encrypted pastes.

End-to-End Encryption

Supported in the web interface for private and unlisted pastes. Your content is encrypted in your browser before upload, and only people with the decryption key can read it.

Want to understand how your data stays private? Learn more about End-to-End Encryption

Public by Default

Every paste is accessible via link.

API Access

Generate API keys and create pastes programmatically.

Python SDK

The official PasteDB Python SDK lets you create and manage pastes with a simple API. It supports both authenticated requests and guest paste creation without requiring an API key. Full documentation and examples are available at Docs Page.

VS Code Extension

Create, upload, and manage your PasteDB snippets directly from Visual Studio Code without leaving your editor.

Custom URLs

Create memorable links for sharing.

Auto Save Drafts

Never lose your work again. PasteDB automatically saves your unfinished paste as a local draft while you type. If you accidentally close the tab or refresh the page, your draft is restored the next time you return.

 
Open a Paste with QR

Scan a PasteDB QR code using your camera or upload a screenshot containing a QR code. Public and encrypted share links are supported automatically.

Open Scanner

  Analytics

Track views and visitors for your pastes.

📡 Nearby Share

Instantly send public pastes to nearby devices. Simply make the receiving device discoverable from the Transfer Page, then tap Nearby Share on the paste to transfer it with a single tap.

Image support

Upload and share images alongside text.

Burn After Read

Create one-time secret pastes that automatically delete themselves after the first successful view. Perfect for sharing passwords, tokens, or sensitive information securely.

  Code Templates

Kickstart your work with built-in templates for HTML, Python, Flask, FastAPI, Django, React, and many more. Click "Start with a Template" in the editor to instantly load a ready-to-use boilerplate and start coding in seconds.

Google Login Control

Login to edit or delete your pastes anytime.

  Run Code

The paste viewer includes a built-in Run Code button for supported languages, letting you execute code instantly without leaving PasteDB.
