Frequently Asked Questions

Where is my data stored?

Paste content and metadata are stored in MongoDB. Uploaded images are stored securely through Cloudinary.

Do I need an account?

No. Public pastes can be created without signing in. Google login is only required for managing your own pastes.

Can I create custom URLs?

Yes. PasteDB supports custom IDs for easier sharing and branding.

Does PasteDB provide an API?

Yes. Developers can generate API keys and create pastes using code. See the API documentation for details.

How can I contribute?

You can contribute by reporting bugs, suggesting features, or opening pull requests on GitHub.
